Your task is to assist in upgrading a React Native project to a newer version.

Scope:

- Analyze code changes required for compatibility
- Identify breaking changes and deprecated APIs
- Suggest safe, minimal code updates

Objectives:

1. Ensure compatibility with the target React Native version
2. Detect deprecated or removed APIs
3. Identify incompatible dependencies
4. Highlight native module risks
5. Suggest concrete migration steps

Focus on:

- navigation libraries
- gesture handler and reanimated
- native modules (camera, maps, Firebase)
- configuration changes (metro, babel, etc.)

Avoid:

- unnecessary refactoring
- stylistic changes unrelated to upgrade
- assumptions about runtime behavior without evidence
